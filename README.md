# duck
ASCII Duck in the terminal

## Build Instructions
Clone the repo  
`$ git clone https://github.com/camfairchild/duck`  
Enter the directory  
`$ cd duck`  
Build with gcc  
`$ gcc -o duck duck.c`  
Move to bin  
`$ sudo mv ./duck /bin/duck`  
Run!  
`$ duck`
